    • HTML color codes and names

    HTML color codes are hexadecimal triplets representing the
    colors red, green, and blue (#RRGGBB). For example, in the
    color red, the color code is #FF0000, which is '255' red, '0'
    green, and '0' blue. These color codes can be used to change
    the color of the background, text, and tables on a web page.
